         final Unit clock(Temporal time, Boolean colonOn, Boolean useAmPm)

        Print a time to the display in hours:minutes. If useAmPm is true, the hours will be displayed as 1-12 with AM/PM. If colon is true, a colon will be displayed between the two values.

      • setDigit

         Unit setDigit(Integer digit, Integer hiValue, Integer lowValue)

        Directly set segment values for the digit in the display. The digit is zero-based, so 0 is the left-most digit. The hiValue is the upper 7 bits of the segment value, and lowValue is the lower 7 bits. The lowValue is not used for 7-segment displays.
