public final class CRICKITHatDeviceFactory
extends AbstractDeviceFactory implements GpioDeviceFactoryInterface, AnalogInputDeviceFactoryInterface, ServoDeviceFactoryInterface, PwmOutputDeviceFactoryInterface

    A device factory for the AdaFruit CRICKIT.

    It is HIGHLY recommended to use the convenience functions instead of creating devices via the builders or constructors as the pin numbering gets a little odd.
